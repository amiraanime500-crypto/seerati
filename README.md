# سيرتي — منشئ سيرة ذاتية عربية

تطبيق وِب يعمل **بالكامل في المتصفح، بلا خادم وبلا قاعدة بيانات**. هذا يعني تكلفة تشغيل = **صفر**، واستضافة مجانية، ولا شيء يتعطّل.

## ما الذي يفعله
- نموذج عربي RTL لبناء سيرة ذاتية: معلومات شخصية، نبذة، خبرات، تعليم، مهارات، لغات.
- **معاينة حيّة** بحجم A4 تتحدّث أثناء الكتابة.
- ٣ قوالب (عصري، كلاسيكي، مدمج) + ألوان + خطوط عربية.
- حفظ تلقائي في المتصفح (localStorage).
- تصدير **PDF** عبر طباعة المتصفح (Ctrl/Cmd+P → حفظ كـ PDF) — لا مكتبات، لا تكلفة.

## الملفات
```
index.html   البنية (RTL، عربي)
styles.css   التصميم + ستايل الطباعة/PDF
app.js       المنطق كله (بلا أي مكتبة خارجية)
```
لا حاجة لـ npm أو build. افتح `index.html` في المتصفح ويعمل فوراً.

## كيف يكسب مالاً (Freemium بدفعة واحدة)
- **مجاني:** بناء السيرة + قالبان + ٣ ألوان + تصدير PDF **مع سطر صغير أسفل الصفحة** (علامة مائية نصية).
- **مدفوع ($2.99 دفعة واحدة):** إزالة العلامة المائية نهائياً + القالب الاحترافي «المدمج» + ألوان إضافية.

### ربط الدفع بصفر بنية تحتية (مهم)
بما أنه لا يوجد خادم، استخدم **بوابة دفع مستضافة** تبيع "كود تفعيل":
1. أنشئ منتجاً على **Gumroad** أو **Lemon Squeezy** (مجاني، يأخذون نسبة فقط عند البيع).
2. فعّل ميزة **License Keys** في المنتج — يولّد لكل مشترٍ كوداً.
3. في `app.js` غيّر `BUY_URL` إلى رابط منتجك.
4. التحقق من الكود في `isValidLicense()`:
   - الحل السريع (يكفي للبداية): نمط ثابت مثل `SEERATI-XXXX`.
   - الحل الصحيح: استدعِ License Verification API لدى Gumroad/Lemon Squeezy (نداء `fetch` واحد من المتصفح، لا يحتاج خادمك).
   - كود تجريبي للاختبار الآن: `DEMO-UNLOCK`.

> ملاحظة صريحة: أي تحقّق في المتصفح وحده يمكن تجاوزه تقنياً. هذا مقبول لمنتج بهذا السعر — معظم الناس يدفعون مقابل الراحة لا القرصنة. لو نمت المبيعات، أضف تحقّقاً عبر Gumroad API.

## النشر مجاناً (٥ دقائق)
**الخيار الأسهل — Netlify Drop:**
1. افتح `app.netlify.com/drop`.
2. اسحب مجلد `seerati` كاملاً إلى الصفحة.
3. ستحصل على رابط مباشر. (اربط نطاقاً خاصاً لاحقاً.)

**أو GitHub Pages:**
1. ارفع الملفات إلى مستودع GitHub.
2. Settings → Pages → اختر الفرع `main`.

كلاهما مجاني تماماً ويتحمّل آلاف الزوّار.

## خطة الوصول لأول ١٠٠ مستخدم — بصفر ميزانية
المبدأ: لا تحارب التوزيع، اجعل المنتج يُكتشف بالبحث ويُشارك نفسه.

1. **تحسين محرّكات البحث (ASO/SEO):** الصفحة مهيّأة بـ `title` و`description` يستهدفان "سيرة ذاتية عربية". أضف لاحقاً صفحات محتوى ("نموذج سيرة ذاتية لخريج"، "سيرة ذاتية للوظائف الحكومية") — كل صفحة تجلب بحثاً.
2. **حلقة المشاركة:** العلامة المائية المجانية تحمل اسم الموقع — كل سيرة مجانية تُرسل = إعلان مجاني.
3. **المجتمعات المجانية:** انشر في مجموعات الوظائف على فيسبوك/تيليجرام/X، ومنتديات الباحثين عن عمل. لا تُعلن — ساعد: "صنعت أداة مجانية لسيرة عربية، جرّبوها."
4. **القياس:** أضف Plausible أو حتى Google Analytics المجاني لتعرف ما الذي يجلب الزوّار وأين يتوقفون.

### الخطوة التالية المنطقية بعد الإطلاق
أضف **خطاب تعريفي (Cover Letter)** و**نسخة إنجليزية للسيرة** — أكثر ميزتين يطلبهما الباحثون عن عمل، وتزيدان قيمة النسخة المدفوعة.
